# Sistema de Gestión de Solicitudes

Aplicación web simple para registrar clientes, crear solicitudes y consultar su estado.

Proyecto desarrollado para el **Desafío 2 – Desarrollo de Software (UDB)**.

---

## Tecnologías

Backend
- Node.js
- Express
- SQLite

Frontend
- HTML
- CSS
- JavaScript

---

## Instalación

Clonar el repositorio
git clone https://github.com/Renderos/DSE_Desafio.git
Entrar al proyecto


cd DSE_Desafio


Instalar dependencias


npm install


---

## Ejecutar el proyecto


node server.js


Abrir en el navegador:


http://localhost:3000
